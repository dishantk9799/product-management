# 🛍️ Product Management
---

## 🚀 Features

- Create a new product  
- Get all products  
- Get a single product by ID  
- Update product details  
- Delete a product  
- MongoDB integration using Mongoose  

---

## 🛠️ Tech Stack

- Backend: Node.js, Express.js  
- Database: MongoDB (Atlas)
